Installation
------------

Create a virtualenv (use ``virtualenvwrapper``): ::

    mkvirtualenv hub


Install environment to development: ::

    make install-dev


Run the project: ::

    make runserver


Tests
-----

To run the test suite, execute: ::

    make test


To show coverage details (in HTML), use: ::

    make test html
